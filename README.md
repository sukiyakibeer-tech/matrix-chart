# プロジェクト名（九星気学アプリ）

## 年盤、月盤表示プログラム（Java）
    - 最近Javaの基礎を学習したばかりで、練習用に作成してみました。
    - 今後は生年月日を入力して本命星、月命星の算出と運勢点数を出すところまで発展させたいと思っています。

### Folder Structure
    - 階層などはなく、App.javaとKigakuMatrix.javaの2つだけです。

### 🚀 機能
    - 1907年以降の任意の年月を入力してください。
    - その入力した年の前後含めた3年間の年盤と入力した月の前後含めた3ヶ月の月盤を表示します
    - 九星気学の年盤、月盤マトリックスは上が北、下が南になります。

### 🛠 使用技術
    - Java 21.0
    - VS Code

### 🏃 実行方法
    1. `src/App.java` を開きます。
    2. VS Codeの実行ボタンを押すか、ターミナルで `App.java` を実行してください。
    3. 九星気学の年盤と月盤を表示したい年（西暦）と月を入力してください。
    4. 入力した年月の前年と翌年の年盤、前月と翌月の月盤を表示します。
